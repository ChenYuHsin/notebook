# object-fit

The **object-fit** CSS property specifies how the contents of a **replaced element**(img, video, svg, iframe...) should be fitted to the box established by its used height and width.

## value

```scss
/* Keyword values */
object-fit: fill;
object-fit: contain;
object-fit: cover;
object-fit: none;
object-fit: scale-down;

/* Global values */
object-fit: inherit;
object-fit: initial;
object-fit: unset;
```
