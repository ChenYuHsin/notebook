## 建立 git project
有本地專案資料夾，並在 github建立好對應之 Repo。
```git
cd 專案資料夾位置

git init 

git add . 

git commit -m "XXXX"

git remote add origin https://github.com/ChenYuHsin/XXXX.git 

git push -u origin master
```
## 創建 gh-pages分支
```git
git branch gh-pages

git chechout gh-pages

改改東西~~~

git add . 

git commit -m "XXXX"

git push -u origin gh-pages
```
